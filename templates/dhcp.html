Bienvenidos a DHCP

Es un servicio que puede utilizarse para asignar dinámicamente direcciones IP únicas y otra información a los sistemas cliente. Puedes asignar la dirección IP, el nombre de dominio, el nombre de host, la puerta de enlace predeterminada y el servidor DNS mediante el servicio DHCP.
